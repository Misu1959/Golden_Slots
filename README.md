# Golden Slots
## Objectives
1. To design a simple and intuitive slot experience suitable for quick play sessions.
2. To recreate the core mechanics of a classic slot machine.
3. To implement win and loss logic based on symbol combinations.

## About
1. This is a casual Android slot machine game developed using Unity.
2. The game focuses on simplicity, smooth gameplay, and mobile-friendly controls. 

## Features
1. Simple tap-to-spin controls
2. Clean and minimal UI
3. Fast spin animations and results
4. Auto Spins
5. Free Spins

## Usage
1. Install the game on an Android device.
2. Launch the app.
3. Set the number of lines.
4. Set the bet
5. Tap the Spin button to spin the slots.
6. View the results after each spin.
7. Repeat and enjoy casual gameplay anytime.


## Download & Instalation
1. Download the latest release from [Releases](https://github.com/Misu1959/Golden_Slots/releases) section.
2. Extract the ZIP file or run the installer.
3. Launch the application no additional configuration needed.
